<h2>🛠️ Installation Steps:</h2>

<p>1. Clone</p>

```
git clone https://github.com/Anzi15/scribi.git
```

<p>2. Open directory</p>

```
cd scribi
```

<p>3. install dependencies</p>

```
npm i
```

<p>4. run dev server</p>

```
npm run dev
```
